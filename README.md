# e-learningPlatform
Διπλωματική Εργασία: Μελέτη, σχεδιασμός και υλοποίηση βάσης δεδομένων για πλατφόρμα παροχής διαδικτυακών μαθημάτων.

Ένα πλήρες SQL schema για πλατφόρμα e-learning. Υλοποίηση με χρήση PostgreSQL, JavaScript & React. Τα μαθήματα θα είναι κυρίως βίντεο.

Database Features (must have):

1. User
  -     Εγγραφή χρηστών - Με roles (student, lecturer, admin)
  -     Διαχείριση προφίλ - Επεξεργασία στοιχείων (όνομα, φωτογραφία)
  -     Reset password - Αποστολή email
2. Courses
  -     Μαθήματα - Τίτλος, περιγραφή, τιμή, κατηγορία
  -     Δομή μαθημάτων - Course -> Section -> Lesson
  -     Περιεχόμενο – Βίντεο/αρχεία PDF
  -     Ιδρύματα - Σχολές
  -     Εξειδικεύσεις μαθημάτων - π.χ. "Data Science", "Digital Marketing"
3. Transactions
  -     Εγγραφή σε μάθημα
  -     Πληρωμές
4. Assessments & Certifications
  -     Online τεστ/εξετάσεις/ερωτηματολόγια - Πολλαπλής επιλογής, χρονόμετρο, …
  -     Πιστοποιητικά - Αποθήκευση τελικού βαθμού και certificate
5. Communication
  -     Μηνύματα

Πλήρης καταγραφή λειτουργικών απαιτήσεων (must have):
 
1. User Management
  -     Ο χρήστης μπορεί να εγγράφεται με email (UNIQUE) και password (min 8 χαρακτήρες)
  -     Ο admin μπορεί να αναθέσει ρόλους (admin, lecturer, student)
  -     Ο χρήστης μπορεί να επεξεργαστεί το προφίλ του (π.χ. όνομα, φωτογραφία)
  -     Ο χρήστης μπορεί να επαναφέρει τον κωδικό του (το σύστημα στέλνει email reset password)
2. Course Management
  -     Ο lecturer μπορεί να δημιουργήσει ένα νέο μάθημα (τίτλος, περιγραφή, τιμή)
  -     Το μάθημα οργανώνεται σε ενότητες (sections) και μαθήματα (lessons)
  -     Ο lecturer μπορεί να επεξεργαστεί ή να διαγράψει ένα μάθημα
  -     Ανέβασμα βίντεο πχ: (max 500MB)
3. Εγγραφές & Πληρωμές
  -     Ο student εγγράφεται σε μάθημα
  -     Πληρωμή μέσω PayPal (αποθήκευση status π.χ. completed)
4. Εξετάσεις & Πιστοποιητικά
  -     Ο student μπορεί να ολοκληρώσει quizzes (αυτόματη βαθμολόγηση)
  -     Έκδοση πιστοποιητικού όταν π.χ. final_grade ≥ 70%
5. Επικοινωνία
  -     Οι χρήστες στέλνουν μηνύματα (lecturer <-> student)
6. Στατιστικά
  -     Το σύστημα να καταγράφει την πρόοδο του μαθητή (completed lessons)
  -     Στατιστικά μαθημάτων
